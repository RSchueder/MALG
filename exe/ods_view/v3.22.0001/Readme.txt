This file contains information about ODS_View 3.22
--------------------------------------------------
Release 3.22.0001 solves a bug when using the Help /
Search menu.

Release 3.22.0000 adds new functionality when showing
longitudinal profiles: if these are sorted on location
the new menu item "Sort according to &Name and distance"
enables a correct sort order on the SOBEK location.

Release 3.22.0000 also solves a bug when using dates
before jan 1, 1900. They are now sorted in correct order.
